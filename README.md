# devHangar - Container driven project environment for organizations

This is devHanger a suite of microservices to provide projects development projects environments. Mainly for universities, schools and other learning organizations.

We provide a set of services to create and manage cloud based project environments for students and developers.

The devHanger project aims to give students a environment to learn and experiment with a choise of organization managed template containers to dynamically create their own learning environment while also providing the organisation with a standardized internal development and hosting plattform for their projects.
